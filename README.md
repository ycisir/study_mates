# Study Mates

Developing a SaaS-based collaborative e-learning platform using Ruby on Rails

---

### Requirements

- Ruby 3.4
- Rails 8.1
- PostgreSQL (Setup required on local machine)
- Redis
- Bundler
- Git

---

### Local setup

```bash
git clone https://github.com/ycisir/study_mates.git
cd study_mates
bundle install
rails db:create db:migrate
rails s
```

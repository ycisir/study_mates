# StudyMates

Building StudyMates
A platform for professionals to connect and collaborate together.
The project aims to make it easier to share resources and improve learning through collaboration.

### Requirements
- Ruby 3.4
- Rails 8.1
- Postgres (Setup required on local machine)
- Redis

### Local setup
```
git clone https://github.com/ycisir/study_mates.git
cd study_mates
bundle install
rails db:create db:migrate
rails s
```
